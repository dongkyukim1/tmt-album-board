# TMT — Taste My Taste 🎵

> 취향을 공유하고 싶은 사람들의 커뮤니티 — **앨범 보드** 목업

일본 마이너 컬처를 공유하는 힙한 커뮤니티 컨셉의 정적 프로토타입입니다.
글은 없고 **앨범 커버**만 올라오며, 커버를 누르면 인스타그램 상세보기 스타일 모달이 열립니다.

## ✨ 핵심 기능: 블라인드 리뷰

> **내 평점·평론을 남겨야 다른 사람의 의견이 열린다.**

- 처음에는 평균 평점이 `?.?`로 잠겨 있고, 다른 멤버들의 평론은 블러 처리됩니다.
- 별점(★ 1~5) + 한 줄 평(최소 10자)을 모두 작성해야 `잠금 해제` 버튼이 활성화됩니다.
- 제출하면 평균 평점이 공개되고, 다른 사람들의 평론이 모두 드러나며, 내 리뷰가 `YOU` 뱃지로 맨 위에 표시됩니다.

솔직한 첫인상을 먼저 남기게 유도하고, 남의 의견에 휩쓸리지 않게 하는 장치입니다.

## 🎨 디자인

- 리소(riso) 프린트 / 진(zine) 감성 — 형광 레드 + 리소 블루 한정 팔레트, 크림 페이퍼, 그레인 텍스처
- 두꺼운 검정 보더 + 오프셋 그림자, Anton / Zen Kaku Gothic 디스플레이 타이포
- 오리지널 SVG 입술·혀 로고 마크

## 🚀 실행

별도 빌드 없이 정적 파일입니다.

```bash
open index.html
# 또는 로컬 서버
python3 -m http.server 8000   # http://localhost:8000
```

## ⚠️ 참고

- 모든 앨범명·아티스트·커버 아트는 **가상**이며, 커버는 CSS로 생성한 추상 그래픽입니다. (실제 앨범 커버 미사용)
- 평가 상태는 메모리에만 저장되어 새로고침 시 초기화됩니다. (영속화는 localStorage로 확장 가능)
